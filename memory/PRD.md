# Odesa Wave (Wave UA) — PRD v2.0

## Огляд
Муніципальний мобільний застосунок для міста Одеси, що поєднує міські сервіси з модулями безпеки узбережжя, транспортною логістикою та інструментами прямої демократії.

## Стек технологій
- **Frontend:** Expo (React Native) з expo-router, react-native-qrcode-svg
- **Backend:** FastAPI (Python) з WebSocket підтримкою
- **Database:** MongoDB
- **AI:** OpenAI GPT-4o (через emergentintegrations + Emergent LLM Key)
- **Інтеграції:** alerts.in.ua (тривоги), Copernicus Marine (еко-моніторинг), WebSocket (SentryNode)

## MVP Функціонал

### 1. Головний дашборд
- Віджети: Транспорт (статус мостів), Безпека, Еко-моніторинг (температура моря), Де Світло (% електропостачання)
- Швидкі дії: QR Квиток, Тривога, Де Світло, Дядя Жора
- Динамічна тема: Sunny Mode / Storm Mode

### 2. Карта безпеки
- Шари: Безпека (мінні зони ДСНС), Еко (стан води на пляжах), Світло (краудсорсинг)
- 5 прибережних зон з рівнями ризику
- Еко-дані: солоність, температура, чистота води

### 3. Транспорт
- 6 маршрутів (трамваї, тролейбуси, маршрутки)
- Купівля QR-квитків з генерацією QR-коду
- Моніторинг 3 мостів (Пересипський, Іванівський, Хаджибейський)

### 4. Послуги
- Каталог 15 муніципальних послуг з категоріями та пошуком
- Категорії: Документи, Платежі, Транспорт, Інфраструктура, Демократія, Безпека, Здоров'я, Освіта, Цифрові

### 5. Чат-бот «Дядя Жора»
- AI-помічник на базі GPT-5.2
- Одеський колорит та гумор
- Допомога з міськими послугами

### 6. SentryNode — Тривога у дворику
- Кнопка паніки з вибором району та адреси
- Сповіщення сусідів та охорони

### 7. «Де Світло»
- Краудсорсингова карта електропостачання
- Звітування про наявність/відсутність світла по районах
- Агрегований % по місту

### 8. Storm Mode
- Автоматична зміна теми при тривозі
- Червоний неон + темний фон
- Пульсуючий банер тривоги

## API Endpoints
| Метод | Шлях | Опис |
|-------|------|------|
| GET | /api/dashboard | Дашборд з віджетами |
| GET | /api/alerts | Статус тривоги |
| POST | /api/alerts/toggle | Перемкнути тривогу |
| GET | /api/transport/routes | Маршрути транспорту |
| POST | /api/transport/ticket | Купити квиток |
| GET | /api/transport/tickets | Мої квитки |
| GET | /api/transport/bridges | Статус мостів |
| GET | /api/coastal/zones | Прибережні зони |
| GET | /api/coastal/eco | Еко-моніторинг |
| POST | /api/light/report | Звіт про світло |
| GET | /api/light/reports | Усі звіти |
| GET | /api/light/summary | Зведення по районах |
| POST | /api/sentry/panic | Тривога у дворику |
| GET | /api/services | Каталог послуг |
| POST | /api/chat | Чат з Дядею Жорою |
